git pull

git stash

O comando git stash salva as alterações sem commit (tanto as preparadas quanto as não preparadas) para uso posterior e as reverte da cópia de trabalho.

Mais informações

git gc

Identifica e exclui arquivos que não são necessários para o repositório. Gc é uma abreviação de "Garbage Collector". É um comando que observa o seu repositório, identifica arquivos e os remove quando julga-os desnecessários.

Git Clone

Git clone é usado para copiar um repositório existente para dentro de um novo diretório local

Git Merge

Git merge é utilizado para mesclar uma branch em outra, juntando seus históricos de desenvolvimento em um só.

git reset

Retorna o repositório para o estado indicado e desfaz os commits posteriores a ele. Para definir até onde deve ser retornado, é necessário informar a tag do commit destino, obtida ao utilizar o git log.

O comando git config é a função conveniente usada para definir valores de configuração do Git em projetos de nível global ou local.
